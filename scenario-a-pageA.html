Page A — The Trap

This page contains a link that opens Page B in a new tab using
rel="opener" — meaning Page B can access window.opener
and redirect this tab while you're not looking.

What's about to happen
  1. Click the link below — Page B opens in a new tab.
  2. You switch focus to the new tab (Page B).
  3. Page B silently redirects this tab to Page C (a fake login).
  4. You switch back here — and see a phishing page instead.
The Vulnerable HTML
<a href="scenario-a-pageB.html" target="_blank" rel="opener"> Visit Page B </a>

rel="opener" explicitly grants the opened tab access to window.opener, allowing it to manipulate this tab's location.
